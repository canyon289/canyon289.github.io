## Better Code Samples ##

This plugin wraps all `table` blocks with a class attribute `.codehilitetable` in an additional `div` of class `.hilitewrapper`. It thus permits to style codeblocks better, especially to make them scrollable.

## Requirements##

* beautifulsoup4
