Using Linux Part 2

Sun 20 August 2017

A couple of months ago I decided to start using Linux as my primary OS. In a prior article I wrote about some tips before you start. Although I've messed around with Linux, and OSX for a number of years, Windows has been my primary OS. No longer is that the case.

Why the switch

I could say that its because I no longer need to use Windows only applications. In the past when I was doing more Mechanical Engineering I would need Computer Aided Design software such as Solidworks to do my job. But I could stop using that and stay with Windows for general computing tasks. I could also say it's because Windows isn't free which was my excuse when I was younger and had less money to pay for things. But it actually is because Windows it not only not fun to use, it actively harms my computing experience.

Linux is lean

Windows takes up more and more resources on my computers every year. There's just so much built in, between drivers, utilities, bloatware installed from other companies. A Linux install of less than one gigabyte accomplishes everything I need.

Linux isn't obtrusive

It seems that in Windows 8 and 10 Microsoft opened the floodgates for annoying icons that live in your application launcher. In Windows 10 I keep getting ads to use OneDrive. While these seem like a little thing the cost of the distractions just add up over time. I don't want my personal device to constantly be trying to force messages on me that I can't control.

Linux doesn't dial home

Windows 10 seems to send back an absurd amount of data about your usage. I'm sure its anonomized and I'm sure Microsoft will say it's to improve the user experience, but in a world where the NSA and CIA seem to be collecting whatever information they want, it's unsettling to have anything more than the bare miniumum sent through the internet.

Linux is just fun

Clearly a personal opinion but I now find myself enjoying using Linux more and more just for the sheer joy of having a computer I can mess around with. I can use it as an extremely efficient blogging device, or dig into the internals and learn more about how computers work. Linux doesn't try to limit me in anyway.

I'm sticking with it

There are some things that are painful, such as configuring multi monitor support, or figuring out why my sound drivers don't work. But these annoyances far outweigh the pains above. I'm enjoying being part of the Linux community and am glad to stay with it as long as I can.
